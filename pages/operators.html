Operatörler

JavaScript İle Operatörler

Her programlama dilinde olduğu gibi JavaScript'te de çeşitli görevler için operatörler kullanılır

Bu görevler genel olarak şunlardır:

  • Aritmetiksel Operatörler
  • Atama Operatörleri
  • Karşılaştırma Operatörleri
  • Mantıksal operatörler

1 - Aritmetiksel Operatörler

Sayılar üzerinde toplama, çıkarma, çarpma, bölme ve mod alma gibi matematiksel işlemleri yapmak için aritmetik operatörleri kullanılır.

Operatör Açıklama Örnek
+ Toplama 5 + 3 = 8
- Çıkarma 5 - 2 = 3
* Çarpma 4 * 2 = 8
/ Bölme 10 / 2 = 5
% Mod (kalan) 10 % 3 = 1
++ Arttırma let x = 1; x++ = 2
-- Azaltma let x = 5; x-- = 4
** Üs alma 2 ** 3 = 8
let sayi = 10; console.log("Toplama: " + (10 + 5)); console.log("Cikarma: " + (10 - 5)); console.log("Carpma: " + (10 * 5)); console.log("Bolme: " + (10 / 5));

Aritmetiksel Operatörlerde işlem önceliği

Operatör önceliği aritmetiksel operatörlerde işlem sırasını Matemetik kuralları belirler.

var sayi = 50 + 5 * 3; console.log(sayi);

Mesela yukarıdaki örnekte matematik kurallarına uygun olarak önce çarpma sonra toplama işlemi yapılmıştır.

Ama hangi işlemin önce yapılacağını parantez işareti kullanarak kendimiz de belirliyebiliriz

var sayi = (50 + 5) * 3; console.log(sayi);

Eğerki kullanılan operatörler kendi içinde önceliğe sahip değil ise işlem soldan sağa doğru yapılır.

var sayi = 50 + 5 - 3; console.log(sayi);

2 - Atama Operatörleri

Değişkenlere değer atamak için atama operatörleri kullanılır. Temel atama operatörü "=" işaretidir. Diğer atama operatörleri, bir işlemi kısaltılmış şekilde yaparak aynı anda hem atama hem de işlem gerçekleştirir.

Operatör Açıklama Örnek
= Eşittir (atama) let x = 10;
+= Toplayarak atama let x = 5; x += 3; (x = 8)
-= Çıkararak atama let x = 5; x -= 2; (x = 3)
*= Çarparak atama let x = 5; x *= 4; (x = 20)
/= Bölerek atama let x = 10; x /= 2; (x = 5)
%= Mod alarak atama let x = 10; x %= 3; (x = 1)
**= Üs alarak atama let x = 2; x **= 3; (x = 8)
let sayiAtama = 5; sayiAtama += 5; console.log("+= operatörü: " + sayiAtama); // Çıktı: 10 sayiAtama -= 3; console.log("-= operatörü: " + sayiAtama); // Çıktı: 7 sayiAtama *= 2; console.log("*= operatörü: " + sayiAtama); // Çıktı: 14 sayiAtama /= 7; console.log("/= operatörü: " + sayiAtama); // Çıktı: 2 sayiAtama %= 3; console.log("%= operatörü: " + sayiAtama); // Çıktı: 2 sayiAtama **= 2; console.log("**= operatörü: " + sayiAtama); // Çıktı: 4

3 - Karşılaştırma Operatörleri

İki değeri karşılaştırmak ve sonucunda bir boolean (true veya false) değer döndürmek için karşılaştırma operatörleri kullanılır. Bu operatörler genellikle koşullu ifadelerde (if, else if, else) ve döngülerde kullanılır.

Operatör Açıklama Örnek Sonuç
== Eşittir (değere göre) 5 == '5' true
=== Kesin eşittir (değere ve türe göre) 5 === '5' false
!= Eşit değildir (değere göre) 5 != '5' false
!== Kesin eşit değildir (değere ve türe göre) 5 !== '5' true
> Büyüktür 5 > 3 true
< Küçüktür 3 < 5 true
>= Büyük veya eşittir 5 >= 5 true
<= Küçük veya eşittir 3 <= 5 true
console.log("5 == '5': " + (5 == '5')); console.log("5 === '5': " + (5 === '5')); console.log("5 != '5': " + (5 != '5')); console.log("5 !== '5': " + (5 !== '5')); console.log("10 > 5: " + (10 > 5)); console.log("3 < 7: " + (3 < 7)); console.log("5 >= 5: " + (5 >= 5)); console.log("3 <= 5: " + (3 <= 5));

4 - Mantıksal Operatörler

Boolean değerleri (true veya false) üzerinde mantıksal işlemler yapmak için mantıksal operatörler kullanılır. Genellikle birden fazla koşulu birleştirmek veya bir koşulun sonucunu tersine çevirmek için kullanılırlar.

Operatör Açıklama Örnek Sonuç
&& VE (AND) - Her iki koşul da doğruysa true döner true && true true
|| VEYA (OR) - Koşullardan en az biri doğruysa true döner true || false true
! DEĞİL (NOT) - Koşulun tersini döner !true false
let kosul1 = 5 > 3; // true let kosul2 = 10 < 20; // true let kosul3 = 4 == 6; // false console.log("kosul1 && kosul2: " + (kosul1 && kosul2)); // Çıktı: true console.log("kosul1 && kosul3: " + (kosul1 && kosul3)); // Çıktı: false console.log("kosul1 || kosul3: " + (kosul1 || kosul3)); // Çıktı: true console.log("!kosul1: " + (!kosul1)); // Çıktı: false console.log("!kosul3: " + (!kosul3)); // Çıktı: true
