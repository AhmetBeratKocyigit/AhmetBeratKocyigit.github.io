2 - Karar Yapıları: if - else

JavaScript'te programın akışını kontrol etmek için if, else if ve else blokları kullanılır. Bu yapılar sayesinde farklı durumlar için farklı işlemler yapılabilir.

if yapısı

if bloğu, belirttiğimiz bir koşul doğru (true) ise çalışır. Yanlış (false) ise atlanır.

let yas = 18; if (yas >= 18) { console.log("Reşitsiniz."); }

Bu örnekte yas değişkeni 18'e eşit ya da büyük olduğu için konsola "Reşitsiniz." yazısı yazdırılır.

if - else yapısı

Koşul doğruysa bir işlem yapılır, değilse başka bir işlem yapılır.

let not = 45; if (not >= 50) { console.log("Geçtiniz."); } else { console.log("Kaldınız."); }

Bu yapıyla tek bir koşulun sağlanıp sağlanmadığına göre farklı işlemler yapılabilir.

else if ile birden fazla durum kontrolü

else if blokları, birden fazla koşulu sıralı şekilde kontrol etmemizi sağlar.

let puan = 85; if (puan >= 90) { console.log("Pekiyi"); } else if (puan >= 75) { console.log("İyi"); } else if (puan >= 50) { console.log("Orta"); } else { console.log("Kötü"); }

Burada program sırayla her koşulu kontrol eder. Uygun koşul bulunduğunda o blok çalışır ve kalanlar atlanır.

Gerçek dünya örneği

let saat = new Date().getHours(); let mesaj; if (saat < 10) { mesaj = "Günaydın"; } else if (saat < 18) { mesaj = "İyi günler"; } else { mesaj = "İyi akşamlar"; } console.log(mesaj);

Bu örnekte kullanıcının saatine göre farklı selamlamalar gösterilir.

İç içe if (Nested if)

Bir if bloğu içinde başka bir if kullanarak daha karmaşık kontroller yapılabilir.

let kullaniciAdi = "admin"; let sifre = "1234"; if (kullaniciAdi === "admin") { if (sifre === "1234") { console.log("Giriş başarılı."); } else { console.log("Şifre yanlış."); } } else { console.log("Kullanıcı adı yanlış."); }

Bu yapı sayesinde çok adımlı doğrulamalar yapılabilir.

Kısa if kullanımı (tek satır)

Eğer sadece tek satırlık bir işlem yapılacaksa süslü parantezler opsiyoneldir:

let hava = "güneşli"; if (hava === "güneşli") console.log("Güneş gözlüğünü unutma!");

Ancak okunabilirlik açısından süslü parantez kullanımı her zaman önerilir.

if yapısında karşılaştırma ve mantıksal operatör kullanımı

if koşullarında genellikle karşılaştırma (==, ===, !=, >, <) ve mantıksal (&&, ||, !) operatörler kullanılır.

let yas = 25; let ehliyet = true; if (yas >= 18 && ehliyet === true) { console.log("Araç kullanabilirsiniz."); }

Bu örnekte hem yaş hem de ehliyet kontrolü yapılmıştır. İki koşul da doğruysa işlem gerçekleşir.
