JavaScript İle Fonksiyonlar

JavaScript'te fonksiyonlar, belirli bir işlemi tekrar kullanılabilir bir şekilde gerçekleştiren kod bloklarıdır.

Fonksiyonlar, genellikle şu görevler için kullanılır:

  • Bir işlemi tekrar etmek
  • Veri işleme ve hesaplama yapmak
  • Modüler kod yazmak

1 - Fonksiyon Tanımlama

JavaScript'te bir fonksiyon, function anahtar kelimesi ile tanımlanır ve belirli bir işlevi yerine getiren kodları içerir.

Özellik Açıklama Örnek
Fonksiyon Tanımlama Fonksiyonun temel tanımı yapılır. function selamla() { console.log("Merhaba!"); }
function selamla() { console.log("Merhaba!"); } selamla(); // Çıktı: Merhaba!

Fonksiyonlara Parametreler Eklemek

Fonksiyonlar, dışarıdan parametreler alabilir ve bu parametreler fonksiyonun işleyişini etkileyebilir.

function selamla(isim) { console.log("Merhaba, " + isim + "!"); } selamla("Ahmet"); // Çıktı: Merhaba, Ahmet!

Fonksiyonlar Değer Döndürebilir

Bir fonksiyon, bir işlem yaptıktan sonra bir değer döndürebilir. Bu değer, return anahtar kelimesi ile belirtilir.

function toplama(a, b) { return a + b; } console.log(toplama(5, 3)); // Çıktı: 8

Ok Fonksiyonları (Arrow Functions)

Ok fonksiyonları, daha kısa bir yazım şekli sunar ve genellikle anonim fonksiyonlar için kullanılır.

const topla = (a, b) => a + b; console.log(topla(2, 3)); // Çıktı: 5

Varsayılan Parametreler

Fonksiyon parametrelerine varsayılan değerler atanabilir. Bu değerler, parametreye bir değer verilmediğinde kullanılır.

function selamla(isim = "Misafir") { console.log("Merhaba, " + isim + "!"); } selamla(); // Çıktı: Merhaba, Misafir! selamla("Ali"); // Çıktı: Merhaba, Ali!

Rest Parametreleri

Rest parametreleri, fonksiyona birden fazla parametre gönderildiğinde, bu parametreleri bir dizi olarak almanızı sağlar.

function topla(...sayilar) { let toplam = 0; for (let sayi of sayilar) { toplam += sayi; } return toplam; } console.log(topla(1, 2, 3, 4)); // Çıktı: 10

Fonksiyon İfadeleri

Fonksiyonlar bir ifadeye de atanabilir. Bu tür fonksiyonlar, genellikle anonim fonksiyonlardır.

const topla = function(a, b) { return a + b; }; console.log(topla(3, 4)); // Çıktı: 7

Fonksiyon İçinde Fonksiyon

Bir fonksiyon içinde başka bir fonksiyon tanımlanabilir. İç fonksiyonlar dış fonksiyonların işleyişini etkileyebilir.

function disFonksiyon() { function icFonksiyon() { console.log("İç fonksiyon çalıştı!"); } icFonksiyon(); // Çıktı: İç fonksiyon çalıştı! } disFonksiyon();

Fonksiyonun this Anahtar Kelimesi

Fonksiyonlarda kullanılan this anahtar kelimesi, fonksiyonun çağrıldığı bağlama bağlı olarak değişir.

function kişi() { this.isim = "Ahmet"; } let obj = new kişi(); console.log(obj.isim); // Çıktı: Ahmet

Callback Fonksiyonları

Callback fonksiyonları, bir fonksiyonun tamamlandığında çalıştırılması için diğer bir fonksiyona parametre olarak verilen fonksiyonlardır.

function yazdir(message, callback) { console.log(message); callback(); } yazdir("Merhaba", function() { console.log("Fonksiyon bitti!"); });
